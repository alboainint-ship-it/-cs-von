About Von Neumann Architecture

The Von Neumann Architecture, designed by John von Neumann, is the foundation of most modern computers. It is also called the stored-program architecture.

Key Idea:

Data and instructions are stored in the same memory. The CPU fetches instructions and data from this memory one at a time.

Main Components:

  • Memory Unit: Stores both instructions and data
  • Control Unit (CU): Directs operations and fetches instructions
  • Arithmetic Logic Unit (ALU): Performs calculations and logic operations
  • Input Unit: Receives data from the outside world
  • Output Unit: Sends results to the user

Fetch–Decode–Execute Cycle:

1. Fetch: Get instruction from memory
2. Decode: Understand the instruction
3. Execute: Perform the instruction

Bottleneck:

Instructions and data share the same memory and bus. The CPU can only access one at a time, which can slow down processing.
